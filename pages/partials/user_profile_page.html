User Profile Page

User can view and update profile information and other data related to his profile on eCommerce MVC. There are multiple tabs for different data.
URL: /profile

  1. Info Tab
    User can see and update profile information such as Full Name, Email, Username, Address etc on this tab.
  2. Change Password Tab
    User can update account password on this tab.
  3. Change Avatar Tab
    User can update avatar (profile photo) on this tab.
  4. Orders Tab
    Users can check their orders history from this tab. Each order displays Placed on Date, Amount, Order Status and Tracking link.
  5. Comments Tab
    Users can also see their comments made on products. Every comment shows the comment description and Rating for the product. Users can also delete their comments from this tab.
