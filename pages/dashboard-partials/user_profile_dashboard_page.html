User Details Page

This is the user details page. This page is similar to the user profile page. The only other tab on this page is Roles tab. Administrator can assign or remove roles from users on Roles tab.
URL: /dashboard/users/userdetails/{user-id}

  1. Info Tab
    User info can be seen and updated such as Full Name, Email, Username, Address etc on this tab.
  2. Roles Tab
    User roles can be modified on this tab.
  3. Orders Tab
    Users orders history can be seen from this tab. Each order displays Placed on Date, Amount, Order Status and Tracking link.
  4. Comments Tab
    Users comments history can be seen from this tab. Every comment shows the comment description and Rating for the product.
